# Chat VeigaGustavo

Interface Flutter (login + chat em tempo real) alinhada ao backend REST e STOMP.

## Sobre o projeto

O **Chat VeigaGustavo** e uma aplicacao Flutter focada em autenticacao e troca de mensagens em tempo real. O app oferece fluxo de login e tela de chat, conectando-se a um backend via API REST (autenticacao e dados iniciais) e STOMP/WebSocket (mensagens em tempo real), servindo como cliente mobile/web para conversas autenticadas.

## Executar

```bash
flutter pub get
flutter run
```

Pacote Dart: `auth_chat_interface` (nome técnico); a app chama-se **Chat VeigaGustavo**.
